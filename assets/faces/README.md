# Face Assets

Place custom face sprite images here (optional).

## Supported Files

- `neutral.png` - Default/idle expression
- `happy.png` - Positive/cheerful expression  
- `thinking.png` - Processing/contemplating
- `listening.png` - Actively listening
- `speaking.png` - While talking
- `surprised.png` - Unexpected input
- `confused.png` - Didn't understand

## Requirements

- PNG format with transparency (recommended)
- Any size (will be auto-scaled)
- Square aspect ratio works best

## Without Custom Sprites

If no sprites are found, the app uses procedural faces drawn with simple shapes - a cute yellow emoji-style face with animated eyes and mouth.

## Getting Sprites

You can create your own using:
- GIMP (free)
- Photoshop
- Microsoft Fluent Emojis (what Max Headbox uses)
- Any emoji/avatar generator

Or use AI image generators to create custom faces!
